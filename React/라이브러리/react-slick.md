# react-slick
✔라이브러리 설치 
```
npm install react-slick

yarn add react-slick
```

✔내장되어있는 css설치
```
npm install slick-carousel

yarn add slick-carousel
```

✔ 사용할 컴포넌트에 import
```
import "slick-carousel/slick/slick.css";
import "slick-carousel/slick/slick-theme.css";
```
